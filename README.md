# Archivio Agende Perpetue

Un'applicazione per trovare anni passati compatibili con un determinato anno futuro (o viceversa), basata sul calendario perpetuo.

## Caratteristiche

- Ricerca anni con lo stesso calendario (stesso giorno d'inizio e stessa caratteristica bisestile).
- Interfaccia in stile retro Windows.
- Calcolo locale immediato.

## Installazione

1. Installa le dipendenze:
   ```bash
   npm install
   ```

2. Avvia in modalità sviluppo:
   ```bash
   npm run dev
   ```

## Tecnologie

- React 19
- Vite
- Tailwind CSS
- Framer Motion (Motion)
- Lucide React
