# Archivio Agende Perpetue

Sei tirchio? Tuo nonno è un accumulatore seriale e non butta via niente? Questa è un'applicazione per trovare anni passati compatibili con un determinato anno futuro (o viceversa), basata sul calendario perpetuo.

In breve ti risparmi 15 euro di agenda... Però sai che figurone sfoggiare una agenda del 2021 nel 2027?


## Caratteristiche

- Ricerca anni con lo stesso calendario (stesso giorno d'inizio e stessa caratteristica bisestile).
- Interfaccia in stile retro Windows.
- Calcolo locale immediato.
- Risparmi qualche palanca tutti gli anni.
- Bellissima perché l'ho fatta io
- Interfaccia nuova di trinca!

## Tecnologie

- React 19
- Vite
- Tailwind CSS
- Framer Motion (Motion)
- Lucide React
